# templates/ — prebuilt full-site templates

Complete, openable site templates (apps, not snippets — exempt from the
snippet INDEX/conventions, like `demo/`). Each template is a self-contained
folder you can copy out wholesale and ship.

| Template | What | Open |
| --- | --- | --- |
| **atelier** | Dependency-free luxury clothing webshop: full-screen hero (image/MP4), editorial category layout, 35-product demo catalogue (4 coordinated webp images each), product pages in 4 selectable layouts (Atelier / Maison / Luisa / Mango), filtering + search + localStorage bag, and a "Store Studio" owner admin (brand, logo, hero, theme colours, 20 fonts + TTF upload). | `templates/atelier/index.html` (serve the folder or open directly) |

Agents: treat a template as a STARTING POINT — copy it, rebrand via its Studio
settings or directly in the markup, swap catalog assets, and apply
`art-direction.skill.md` for imagery/motion. Not in the npm package (assets are
heavy); lives in the repo only.
